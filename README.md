# Avenure - Tecnologia Transparente

Landing page da Avenure, softhouse especializada em soluções de software.

## Stack

- **Vue 3** + Composition API
- **TypeScript**
- **Tailwind CSS**
- **Vite**
- **Vue Router**

## Desenvolvimento

```bash
npm install
npm run dev
```

Acesse [http://localhost:5173](http://localhost:5173)

## Build

```bash
npm run build
```
